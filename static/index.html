    FAQ

    这是什么？

    这是一个 API 中转服务。你可以用统一的地址和 Key 访问多家 AI 模型，无需分别注册各家服务。

    支持 streaming 吗？

    支持。在请求中加入 "stream": true 即可获得流式响应。

    和直接调用官方 API 有什么区别？

    对你的代码来说没有区别——请求和响应格式完全一样，只是换了地址和 Key。

    API Key 安全吗？

    你的 Key 只用于本服务的身份验证，不会被发送到任何上游服务商。请妥善保管，不要分享给他人。
